# Screenshots Folder

This folder is for storing application screenshots for the README.

## Recommended Screenshots

1. **dashboard.png** - Main dashboard view showing statistics
2. **lead-list.png** - List of all leads with filters
3. **lead-detail.png** - Individual lead detail page with notes
4. **create-lead.png** - Create lead modal
5. **login.png** - Login page (optional)

## How to Take Screenshots

1. Run the application (`npm run dev:full`)
2. Create some sample leads
3. Navigate to each page
4. Take screenshots (Windows: Win + Shift + S, Mac: Cmd + Shift + 4)
5. Save them to this folder
